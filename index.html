Configuration d'eM Client

1
Téléchargez et installez eM ClientRendez-vous sur emclient.com et téléchargez la version adaptée.
2
Ajoutez votre compte emailLancez eM Client, cliquez sur Menu > Comptes > Ajouter un compte.
3
Configurez les paramètres du serveurRemplissez les détails du serveur entrant (IMAP) et sortant (SMTP).
Onglet: IMAP
  • Hôte : mail.adexcloud.dz
  • Port : 143
  • Sécurité : Forcer SSL/TLS
Onglet: SMTP
  • Hôte : smtp.adexcloud.dz
  • Port : 587
  • Sécurité : SSL/TLS si disponible

Étapes de configuration

1
Ouvrez les paramètresAccédez aux paramètres de compte dans votre client de messagerie.
2

3
Configurez le serveur IMAPServeur: mail.adexcloud.dz — Port: 993 — Chiffrement: SSL/TLS

5
Testez la connexionEnvoyez un email test pour vérifier que la configuration fonctionne.
